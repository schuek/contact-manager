# Gestor de Contactos
Aplicación CRUD hecha con HTML, CSS y JavaScript que permite crear, editar, eliminar y listar contactos.

## Tecnologías
- HTML
- CSS
- JavaScript (DOM)
- LocalStorage

## Funcionalidades
- Crear contactos
- Editar contactos
- Eliminar contactos
- Buscar por nombre

## Aprendizaje
Este proyecto me ayudó a consolidar manipulación del DOM y almacenamiento local.
